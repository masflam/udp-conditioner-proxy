# UDP Conditioner Proxy

Simple UDP proxy to simulate network conditions, with a GUI.

This is a fork of [pierd/udp-conditioner-proxy](https://github.com/pierd/udp-conditioner-proxy)
with an [egui](https://github.com/emilk/egui)-based GUI for controlling parameters and
starting/stopping the proxy at runtime, on top of the original CLI-only tool.
